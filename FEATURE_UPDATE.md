# 🎉 功能更新：声音克隆 + 多语言支持

## ✨ 新增功能

### 1. 🎤 声音克隆功能

**全新声音克隆系统**，让翻译后的语音使用您自己的声音！

#### 主要特性
- ✅ **个性化TTS输出**：使用您克隆的声音进行翻译
- ✅ **简单配置**：只需在控制台训练一次，获取Speaker ID即可
- ✅ **持久保存**：配置自动保存，无需重复输入
- ✅ **可视化管理**：专门的声音克隆对话框

#### 使用方法
1. 在主界面点击 **"🎤 声音克隆"** 按钮（紫色）
2. 输入您从豆包控制台获取的 Speaker ID
3. 点击保存
4. 正常使用翻译功能，TTS自动使用您的声音

#### 界面更新
```
[▶ 开始] [■ 停止] [🗑 清空] [⚙ 设置] [🎤 声音克隆] ✨ 声音克隆: voice_xxx...
```

## 🚀 使用示例

### 示例1: 中文→英语（使用声音克隆）

```python
# 1. 配置声音克隆
# 在GUI中点击"🎤 声音克隆"，输入: voice_english_12345

# 2. 配置翻译
源语言: 中文(zh)
目标语言: 英语(en)
模式: s2s (语音到语音)

# 3. 开始翻译
说: "大家好，我是张三"
听: "Hello everyone, I'm Zhang San" (用您的声音)
```

### 示例2: 日语→法语

```python
# 不使用声音克隆，使用默认TTS
源语言: 日语(ja)
目标语言: 法语(fr)
模式: s2s

说: "こんにちは" (日语)
听: "Bonjour" (法语，默认声音)
```

### 示例3: 多语言会议

```python
# 场景：国际团队会议
# 中文→英语（自己发言）
配置: zh → en, speaker_id: voice_en_xxx

# 英语→中文（听对方）
快速切换: en → zh, 无需speaker_id
```
