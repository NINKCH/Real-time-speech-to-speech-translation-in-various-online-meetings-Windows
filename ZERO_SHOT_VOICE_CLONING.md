# 🎤 零样本声音复刻说明

## ✨ 好消息：声音复刻已自动启用！

豆包同声传译2.0模型**内置零样本声音复刻功能**，无需任何额外配置！

## 🎯 什么是零样本声音复刻？

**零样本（Zero-Shot）** = 不需要提前训练
**实时复刻** = 边说话边学习音色

### 传统声音复刻 vs 零样本声音复刻

| 特性 | 传统方式 | 零样本（同传2.0） |
|------|----------|-------------------|
| **准备工作** | ❌ 需要提前录音20-30分钟 | ✅ 无需准备 |
| **训练时间** | ❌ 需要等待几小时 | ✅ 即时生效 |
| **Speaker ID** | ❌ 需要获取和配置 | ✅ 不需要 |
| **使用方式** | ❌ 复杂配置 | ✅ 自动启用 |
| **音色学习** | ❌ 提前采样 | ✅ 实时采样 |
| **效果** | ⭐⭐⭐⭐⭐ 高质量 | ⭐⭐⭐⭐⭐ 高质量 |

## 🚀 如何使用？

### 超简单！只需两步：

1. **选择S2S模式**（语音到语音）
   ```
   模式: [s2s (语音到语音) ▾]
   ```

2. **开始翻译**
   - 点击【▶ 开始翻译】
   - 对着麦克风说话
   - 系统自动学习你的音色
   - 翻译时自动使用你的声音

就这么简单！✨

## 🎭 工作原理

```
你说话（中文）
    ↓
系统实时采样你的音色 🎤
    ↓
语音识别 + 翻译 🧠
    ↓
用你的音色合成翻译结果 🔊
    ↓
输出翻译语音（英文，但是你的声音！）
```

### 技术细节

```python
# 当前配置
mode = "s2s"  # 语音到语音模式

# 这个模式会：
# 1. 实时分析你的语音输入
# 2. 自动提取音色特征
# 3. 在翻译时应用这些特征
# 4. 输出"用你的声音说英文"的效果
```

## 📊 两种模式对比

### S2T模式（语音到文本）

```
你说：你好，我是张三
      ↓
输出：Hello, I'm Zhang San（文本）
```

### S2S模式（语音到语音）✨ 推荐

```
你说：你好，我是张三（用你自己的声音）
      ↓
输出：Hello, I'm Zhang San（音频，但是你的声音！）
```

## 💡 实际效果

### 场景：国际视频会议

**没有声音复刻**（使用默认TTS）：

```

                    ↑ 你的声音！听起来像真人在说英文
```

## ⚙️ 配置说明


## ❓ 常见问题

### Q1: 需要提前录制声音样本吗？

**A:** ❌ **不需要！** 这就是"零样本"的意思。系统会从你实时说话中学习。

### Q2: 需要配置Speaker ID吗？

**A:** ❌ **不需要！** Speaker ID是用于传统声音复刻的。同传2.0自动处理。

### Q3: 需要训练声音模型吗？

**A:** ❌ **不需要！** 没有训练过程，即开即用。

### Q4: 什么时候开始复刻我的声音？

**A:** ✅ **从你开始说话的第一秒！** 系统实时学习，越说越准。

### Q5: 如果我停止说话一会儿，会忘记我的声音吗？

**A:** ✅ **不会！** 系统会记住你的音色特征，直到会话结束。

### Q6: 多人同时说话怎么办？

**A:** ✅ **支持多人！** 系统可以识别并复刻不同说话人的声音。

### Q7: 效果和传统声音复刻相比如何？

**A:** ✅ **同样高质量！** 而且更方便，无需等待训练。

## 🎯 使用建议

### 最佳实践

```
✅ DO:
- 使用S2S模式（默认）
- 在安静环境使用
- 说话清晰，不要太快
```
