# 🔬 声音复刻训练功能更新

## 📋 更新概述

在原有的声音克隆配置功能基础上，新增了**内置声音复刻训练工具**，用户现在可以直接在应用中上传音频训练声音复刻模型，无需完全依赖控制台操作。

## 🔧 技术实现

### 模型类型支持

```python
MODEL_TYPE_MEGA = 0        # 不推荐
MODEL_TYPE_ICL_1_0 = 1     # ICL 1.0
MODEL_TYPE_DIT_STANDARD = 2  # DiT标准版
MODEL_TYPE_DIT_RESTORE = 3   # DiT还原版
MODEL_TYPE_ICL_2_0 = 4     # ICL 2.0 (推荐)
```

### 语种支持

| 代码 | 语言 | 模型支持 |
|------|------|----------|
| 0 | 中文 | 所有 |
| 1 | 英语 | 所有 |
| 2 | 日语 | 所有 |
| 3 | 西班牙语 | model_type≤2 |
| 4 | 印尼语 | model_type≤2 |
| 5 | 葡萄牙语 | model_type≤2 |
| 6 | 德语 | model_type=2 |
| 7 | 法语 | model_type=2 |

### 训练流程

```python
# 1. 创建训练器
trainer = VoiceCloneTrainer(app_key, access_token)

# 2. 设置回调
trainer.on_progress = lambda msg: print(msg)
trainer.on_success = lambda msg: print(msg)
trainer.on_error = lambda msg: print(msg)

# 3. 上传训练
result = trainer.upload_and_train(
    speaker_id="S_xxxx",
    audio_file_path="my_voice.wav",
    model_type=VoiceCloneTrainer.MODEL_TYPE_ICL_2_0,
    language=VoiceCloneTrainer.LANGUAGE_CN,
    enable_denoise=False,
    resource_id="seed-icl-2.0"
)

# 4. 查询状态
status = trainer.check_status("S_xxxx")
```

### 异步训练

使用`QThread`实现异步训练，避免阻塞GUI：

```python
class TrainingThread(QThread):
    progress_signal = pyqtSignal(str)
    success_signal = pyqtSignal(str)
    error_signal = pyqtSignal(str)
    
    def run(self):
        trainer = VoiceCloneTrainer(...)
        result = trainer.upload_and_train(...)
```

## ⚙️ 配置要求

### 必需配置

在`.env`文件中：
```bash
DOUBAO_APP_KEY=your_appid
DOUBAO_ACCESS_KEY=your_access_token
```

### 音频文件要求

- **格式**: WAV, MP3, OGG, M4A, AAC, PCM
- **大小**: 最大10MB
- **时长**: 建议20秒以上
- **质量**: 清晰无噪音，单通道
- **采样率**: 建议16kHz或24kHz

### Speaker ID获取

1. 访问控制台：https://console.volcengine.com/speech/service
2. 购买声音复刻服务
3. 获取 Speaker ID（格式：S_开头）

## ⚠️ 注意事项

### 1. 费用说明
- ✅ 每个 Speaker ID 有训练次数限制
- ✅ 训练需要消耗配额
- ⚠️ 超出配额需要额外购买

### 2. 训练时间
- ⏱️ 通常 3-10 分钟
- ⏱️ 取决于服务器负载
- ⏱️ 训练期间可以关闭对话框

### 3. 音频质量
- ✅ 清晰无噪音的音频效果最好
- ⚠️ 有噪音建议开启降噪
- ⚠️ 降噪可能影响声音细节

### 4. 模型选择
- **ICL 2.0**: 最新，推荐使用
- **ICL 1.0**: 较早版本
- **DiT标准版**: 还原音色，不还原风格
- **DiT还原版**: 还原音色和风格（口音、语速）

## 🐛 故障排除

### 问题1: 上传失败

**可能原因**：
- 文件过大（>10MB）
- 格式不支持
- 网络问题

**解决方案**：
```bash
# 检查文件大小
ls -lh my_voice.wav

# 压缩音频（使用ffmpeg）
ffmpeg -i input.wav -ar 16000 -ac 1 output.wav
```

### 问题2: 训练失败

**错误码说明**：
- `1101`: 音频上传失败
- `1102`: ASR转写失败
- `1103`: 声纹检测失败
- `1104`: 声纹相似度过高
- `1109`: 音频与文本对比字错率过高

**解决方案**：
- 检查音频质量
- 确认语种设置正确
- 重新录制更清晰的音频

### 问题3: Speaker ID无效

**检查**：
- ✅ 格式是否正确（S_开头）
- ✅ 是否已在控制台购买
- ✅ 是否已激活

### 问题4: 训练进度卡住

**处理**：
- 等待几分钟（可能是服务器忙）
- 关闭对话框后重新打开
- 在控制台查看状态

```
✅ DO:
- 使用高质量麦克风录音
```
