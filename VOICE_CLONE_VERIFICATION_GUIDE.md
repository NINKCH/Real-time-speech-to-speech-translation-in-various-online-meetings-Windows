# 🔬 零样本声音复刻验证指南

## 📖 功能说明

声音复刻验证工具用于测试豆包同传2.0的零样本声音复刻功能，让您能够：

- ✅ **验证复刻效果** - 听听是否真的复刻出了您的声音
- ✅ **保存音频样本** - 保存复刻的音频供对比分析
- ✅ **实时测试** - 边说话边采样，即时体验零样本复刻

## 🚀 使用方法

### 步骤1: 打开验证工具

1. 启动应用
   ```bash
   python main.py
   ```

2. 点击主界面的 **【🔬 验证复刻】** 按钮（橙色）
   - 或者点击 **【ℹ️ 功能说明】** 后选择 **【验证复刻】**

### 步骤2: 开始测试

1. **点击【🚀 开始测试】**
   - 确保麦克风已连接并授权
   - 确保网络连接正常

2. **朗读测试文本**（约10秒）
   ```
   你好，我正在测试豆包同传的零样本声音复刻功能。
   这个技术可以实时学习我的声音，并用我的音色说英文。
   让我们看看效果如何。
   ```
   
   💡 **提示**：
   - 清晰地朗读
   - 保持正常语速
   - 不要太快或太慢

3. **等待处理**
   - 系统会实时翻译您的中文
   - 自动学习您的音色
   - 用您的声音合成英文翻译

### 步骤3: 听取结果

1. **测试完成后**
   - 界面会显示"✅ 复刻成功！"
   - 显示音频长度和采样率信息

2. **点击【▶️ 播放复刻音频】**
   - 仔细听：这是您的声音吗？
   - 对比原声：音色是否相似？
   - 注意细节：语调、音高、节奏

### 步骤4: 保存样本（可选）

1. **点击【💾 保存样本】**
   - 选择保存位置
   - 默认文件名：`voice_clone_test_YYYYMMDD_HHMMSS.wav`

2. **使用音频播放器打开**
   - Windows Media Player
   - VLC Media Player
   - 或任何其他播放器

3. **对比分析**
   - 与原始录音对比
   - 评估复刻质量
   - 记录测试结果

## 🔍 工作原理

### 实时复刻流程

```
┌─────────────────────────────────────────┐
│  1. 开始录音（10秒）                     │
│     你朗读测试文本                       │
│            ↓                             │
│  2. 实时发送到豆包同传2.0                │
│     音频流式传输                         │
│            ↓                             │
│  3. 端到端处理                           │
│     ┌────────────────────┐              │
│     │ 实时音色提取        │              │
│     │ 语音识别（中文）    │              │
│     │ 翻译（中→英）       │              │
│     │ 应用复刻音色        │              │
│     │ 语音合成（英文）    │              │
│     └────────────────────┘              │
│            ↓                             │
│  4. 返回复刻音频                         │
│     用您的声音说英文                     │
│            ↓                             │
│  5. 播放和保存                           │
│     验证复刻效果                         │
└─────────────────────────────────────────┘
```

### 关键技术点

1. **零样本学习**
   - 不需要提前训练
   - 边说话边学习音色
   - 实时提取特征

2. **端到端架构**
   - 单一模型处理全流程
   - 避免误差累积
   - 降低延迟

3. **音色复刻**
   - 提取音高特征
   - 保留音色特点
   - 应用到合成

### 常见问题处理

**Q1: 复刻的声音不像我？**

可能原因：
- 录音质量差
- 环境噪音大
- 麦克风距离不当
- 说话不自然

解决方法：
- 改善录音环境
- 调整麦克风位置
- 自然地朗读
- 多次测试对比

**Q2: 没有收到音频？**

检查项：
- 网络连接是否正常
- API配置是否正确
- 麦克风是否授权
- 查看错误日志

**Q3: 音频质量不好？**

改进方法：
- 使用更好的麦克风
- 确保安静环境
- 说话更清晰
- 保持稳定距离

**Q4: 保存的文件无法播放？**

解决方法：
- 确认文件完整性
- 尝试不同播放器
- 检查文件大小
- 重新进行测试
